# Video Game Sales - Exploratory Data Analysis

## Dataset
Video Game Sales dataset containing global sales data across platforms, genres, publishers and regions.

## Objectives
- Identify best-selling genres and platforms over the years
- Analyze the impact of publishers on sales performance
- Track sales trends across regions (NA, JP, EU, Other)
- Compare sales performance of PlayStation, Xbox and Wii
- Guide marketing strategies by targeting high-demand regions

## Tools Used
Python, pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook

## Key Insights
- Action and Sports genres consistently dominated global sales
- Nintendo, EA and Activision were the top performing publishers
- North America accounted for the largest share of global game sales
